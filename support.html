About Clawin

Clawin is a companion mobile application for remotely controlling your local OpenClaw AI agent instance. It connects to your self-hosted OpenClaw server through a relay, allowing you to chat with your AI agent on the go.

Clawin is independently developed and is not affiliated with the open-source OpenClaw project.

Getting Started

  1. Install the OpenClaw plugin on your server by running: openclaw plugins install openclaw-app
  2. Enable the plugin: openclaw config set channels.openclaw-app.enabled true
  3. Open Clawin on your mobile device and tap Connect.
  4. The app will automatically pair with your OpenClaw instance via the relay server.

System Requirements

  • iOS: iOS 16.0 or later
  • Network: An active internet connection is required to communicate with your OpenClaw instance via the relay server.
  • OpenClaw Server: A running OpenClaw instance with the openclaw-app plugin installed and enabled.

Frequently Asked Questions

Q: The app cannot connect to my OpenClaw instance.

Please check the following:

  • Ensure your OpenClaw instance is running.
  • Verify the openclaw-app plugin is installed and enabled.
  • Check your internet connection on both the server and mobile device.
  • If you are using a custom relay server, make sure the URL is correct in Settings.

Q: How do I change the relay server?

Go to Settings → Relay Configuration and enter your custom relay server URL, Room ID, and Auth Token.

Q: Is my data stored on your servers?

No. All connection credentials and chat history are stored locally on your device. Messages are transmitted through the relay server you configure, and we do not access or store any message content. For more details, see our Privacy Policy.

Q: How do I reset the app?

You can delete and reinstall the app to reset all settings and local data. Alternatively, go to Settings and reconfigure your connection.

Known Limitations

  • Clawin requires a self-hosted OpenClaw instance; it does not provide AI capabilities on its own.
  • Push notifications are not currently supported; the app must be open to receive messages.
  • Image and file attachments are subject to relay server size limits.

关于 Clawin

Clawin 是一款配套移动应用，用于远程控制您本地的 OpenClaw AI 智能体实例。它通过 Relay 中继服务器连接到您自托管的 OpenClaw 服务器，让您随时随地与 AI 智能体对话。

Clawin 为独立开发项目，与开源项目 OpenClaw 没有直接关联。

快速开始

  1. 在您的服务器上安装 OpenClaw 插件： openclaw plugins install openclaw-app
  2. 启用插件： openclaw config set channels.openclaw-app.enabled true
  3. 在手机上打开 Clawin，点击 Connect（连接）。
  4. App 将通过 Relay 中继服务器自动与您的 OpenClaw 实例配对。

系统要求

  • iOS：iOS 16.0 或更高版本
  • 网络：需要有效的互联网连接，以通过 Relay 服务器与 OpenClaw 实例通信。
  • OpenClaw 服务器：一个正在运行的 OpenClaw 实例，且已安装并启用 openclaw-app 插件。

常见问题

Q：App 无法连接到我的 OpenClaw 实例。

请检查以下事项：

  • 确认您的 OpenClaw 实例正在运行。
  • 确认 openclaw-app 插件已安装并启用。
  • 检查服务器和手机的网络连接是否正常。
  • 如果使用了自定义 Relay 服务器，请在「设置」中确认 URL 是否正确。

Q：如何更换 Relay 中继服务器？

进入 设置 → Relay 配置，输入您自定义的 Relay 服务器地址、房间 ID 和认证令牌。

Q：我的数据会存储在你们的服务器上吗？

不会。所有连接凭证和聊天记录均存储在您的本地设备上。消息通过您自行配置的 Relay 服务器进行传输，我们不会访问或存储任何消息内容。详情请参阅我们的隐私政策。

Q：如何重置 App？

您可以删除并重新安装 App 来重置所有设置和本地数据。也可以进入设置页面重新配置连接。

已知限制

  • Clawin 需要自托管的 OpenClaw 实例，本身不提供 AI 能力。
  • 目前不支持推送通知；收消息时需保持 App 处于打开状态。
  • 图片和文件附件受 Relay 服务器大小限制。
